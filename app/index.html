VASSET

This is as Store Application for Nexstreaming's Editor SDK Customers LG, TINNO, ZTE. Users can download assets that is used editing by thier Editor Application

Now published LGE and TINNO to Google PlayStore.


This application was developed only one Developer. I made all of module Download to Install service even ICP (AIDL between Editor SDK)


It is not stand alone application. So it does not showing on Launcher App. And This Application must be launched Video editor of LGE or TINNO.


I am not used to Android Databinding. So used MVC,MVP pattern.

LG, TINNO, ZTE 단말 제조사 대상으로 Nextreaming의 동영상 편집 Editor SDK가 판매되어 해당 SDK에사 사용 할 수 있는 편집 아이템 요소를 구매 할 수 있는 스토어 앱이다.

Google PlayStore에는 LGE와 TINNO 에서 사용 할 수 있는 앱이 등록되어있다.


개발자 단독으로 개발을 완료한 앱이며, 기본 UI에서 부터 Download와 Install Service, 고객사 Editor 간의 IPC 통신 (AIDL)까지 모두 개발하였다.


독립적으로 실행되지 않는 Hidden App이기 때문에 Launcher App에 노출되지 않으며, 반드시 LGE의 Video Editor 혹은 TINNO의 Video Editor에 의해 실행되어야 한다.


Android Databinding이 익숙하지 않아 MVC,MVP 패턴을 이용해 디자인 되어있다.

QUICK-ZZAL

QUICK-ZZAL is personal project for personal purpose.

QUICK-ZZAL - Search images was published to Google PlayStore.


It can send images from Google image searching results. I think that is very interesting using when chatting others. Even it can search image by a keyword very faster.

I did Planning, Design, Develope

Searcing image by Google Image search. The results are stored by Realm. Even implements results paging.

The results are dispalyed by Glide. So QUICK-ZZAL can shows GIFs. Retrofit uses getting Google GeoLocation, that location information will be used AdMob.

Firebase uses only Analytics.


개인 프로젝트로 제작되어 배포된 앱이다.

Google PlayStore에는 퀵짤 - 이미지검색, 사진검색으로 현재 등록되어있다.


구글 이미지 검색 결과로 나오는 필요한 이미지를 채팅 중 상대방에게 전달 할 수 있다. 또한 특정 키워드로 원하는 이미지를 빠르게 찾을 수 있다.

기획, 디자인, 개발 모두 담당하였다.

Google Image 검색 엔진을 사용하며, 검색 결과는 Realm을 이용하여 저장한다. Google Image검색 시 제공하는 Paging 기능 또한 구현되어있다.

이미지 검색 결과는 Glide로 표현하여 GIF도 표현된다. Retrofit은 Google GeoLocation API를 사용하기 위해 사용되었으며, Location 정보는 AdMob에 사용된다.

Firebase는 Analytics에 사용 되었다.

장바구니 메모

MARKET MEMO is personal project for personal purpose.

MARKET MEMO was published to Google PlayStore.


MARKET MEMO is made by my wife's requesting. Based QUICK-ZZAL searche images. So user input some product data, It gets product image automatic.

I did Planning, Design, Develope

Stored products are calculate and showing statitics using graph.

Even it can exports stored datas to Excel or CVS, text. Exporting excel uses Apache Poi library. Exporting CVS uses Open CVS library.

Statitics datas are shown GraphView library.

Retrofit uses getting Google GeoLocation, that location information will be used AdMob.

Firebase uses only Analytics.


개인 프로젝트로 제작되어 배포된 앱이다.

Google PlayStore에는 장바구니 메모으로 현재 등록되어있다.


장바구니 메모는 아내의 요청에 의해 제작되었다. QUICK-ZZAL에서 구현한 이미지 검색 엔진을 기반으로 만들어져, 원하는 상품 메모를 입력 할 경우 상품에대한 이미지를 자동으로 검색한다.

기획, 디자인, 개발 모두 담당하였다.

장바구니에 담겨진 상품들을 자동으로 금액을 계산해 주며, 통계를 통해 소비된 장소와 물건들을 한눈에 보여준다.

상품 구매결과를 Excel이나 CVS 또는 일반 텍스트로 내보내기가 가능하다. Excel 내보내기는 Apache Poi가 사용되었으며 CVS는 Open CVS 라이브러리가 사용되었다.

통계에 사용된 Data는 GraphView 라이브러리가 사용되었다.

Retrofit은 Google GeoLocation API를 사용하기 위해 사용되었으며, Location 정보는 AdMob에 사용된다.

Firebase는 Analytics에 사용 되었다.
